This project classifies handwritten digits (0-9) using the MNIST dataset.

##Technologies Used
-Python
-Tensorflow
-Keras
-Numpy
-Matplotlib
-Seaborn


#Features
-Data preprocessing
-ANN model
-Confusion matrix
-Training Accuracy/Loss Visualization


#Results
Achieved high accuracy on the MNIST dataset.
