---

## 📌 Project Overview

The MNIST dataset is one of the most well-known benchmarks in deep learning.
This project builds a Convolutional Neural Network (CNN) from scratch to
classify handwritten digits with high accuracy using TensorFlow and Keras.

## 🎯 Results

| Metric | Value |
|--------|-------|
| Test Accuracy | **99.41%** |
| Test Loss | ~0.02 |
| Dataset | MNIST (70,000 images) |
| Training Images | 60,000 |
| Testing Images | 10,000 |
| Classes | 10 (Digits 0–9) |
